# fact-level
fact level एक ऐसी वेबसाइट है जहाँ कहानियों के ज़रिए ज़िंदगी की सच्चाई दिखाई जाती है। यहाँ गरीबी और अमीरी, खाना-पीना, सेहत और समाज से जुड़ी लॉजिकल और सच्ची कहानियाँ पढ़ने को मिलेंगी। हर कहानी सोचने पर मजबूर करती है और ज़िंदगी को नए नज़रिए से देखने का मौका देती है। आने वाले समय में यहाँ और भी क्रिएटिव व रियल स्टोरीज़ शेयर की जाएँगी। #factlevel
